Root cause: runPredictionScan() was using getUniverseAsync() which
returns 5221 stocks from Supabase (search/autocomplete universe).
This caused Live Predictions tab to show totalScanned=5221.

Fix: switched runPredictionScan() to use NSE_STOCK_UNIVERSE directly
(434 curated stocks), consistent with AI Intelligence, Multibagger,
and UltraQuant engines.

Also removed 6 duplicate entries from NSE_STOCK_UNIVERSE (440 → 434):
MARICO, MUTHOOTFIN, AMBUJACEM, ATUL, DELHIVERY, KANSAINER.

All scan engines now consistently show universe=434.
