fix: include today's candle for 1D interval — prices now consistent across timeframes

Root cause: Upstox historical endpoint never returns today's in-progress candle
for ANY interval (not just intraday). So 1D was showing Friday's close (1414)
while 5M was showing today's live close (1394) — a visible price mismatch.

Fix: extend needsIntradayToday to cover day interval too (only week/month excluded).
Now 1D fetches past days via historical + today's daily candle via intraday endpoint.

Verified: HDFC Bank 1D shows 23/3/2026 close 754.55, 5M shows 754.25 — consistent.
